1. Pendahuluan

Pondok Pesantren Mambaul Ulum (selanjutnya disebut "Pengelola" atau "Kami") menghormati privasi seluruh pengguna aplikasi Ammu Online (selanjutnya disebut "Aplikasi"). Kebijakan Privasi ini menjelaskan jenis data yang dikumpulkan, bagaimana data digunakan, disimpan, dan dilindungi.

Dengan menggunakan Aplikasi, Anda menyatakan telah membaca, memahami, dan menyetujui Kebijakan Privasi ini. Apabila Anda tidak setuju, mohon untuk tidak menggunakan Aplikasi.

2. Data yang Kami Kumpulkan

Kami mengumpulkan data berikut hanya jika diperlukan untuk operasional pondok pesantren:

2.1 Data Identitas

  • Nama lengkap santri, guru, dan pengurus
  • Jenis kelamin
  • Tanggal lahir (jika diisi pada profil)
  • Nomor WhatsApp / nomor telepon
  • Alamat email (untuk login Google Sign-In opsional)
  • Foto profil (opsional, diunggah oleh pengguna sendiri)
  • Tanda tangan digital (opsional, untuk rapor & slip gaji)

2.2 Data Akademik & Operasional

  • Data lembaga, kelas, jabatan
  • Prestasi Qiraati bulanan (level awal, akhir, total)
  • Nilai diniyah, rapor semester
  • Catatan absensi santri dan guru
  • Kalender pendidikan & jadwal kegiatan

2.3 Data Keuangan (terbatas pada Pengurus Keuangan)

  • Tagihan SPP, Syahriyah, Infaq
  • Transaksi pembayaran POS pondok
  • Mutasi tabungan santri & guru
  • Slip gaji pengurus / guru
  • Catatan hutang & arus kas

Aplikasi TIDAK menyimpan nomor kartu kredit, nomor rekening bank, atau detail instrumen pembayaran elektronik apa pun. Semua transaksi keuangan dicatat dalam bentuk nominal rupiah dan kategori, bukan instrumen pembayaran.

2.4 Data Teknis

  • Token notifikasi (FCM token) untuk mengirim notifikasi push
  • Versi aplikasi yang digunakan
  • Log audit perubahan data (siapa mengubah apa, kapan)

2.5 Data yang TIDAK Kami Kumpulkan

  • Lokasi GPS perangkat
  • Daftar kontak telepon
  • Riwayat browsing di luar Aplikasi
  • Mikrofon, kamera (kecuali pengguna memilih unggah foto)
  • Data biometrik (sidik jari, wajah)
  • Iklan personalisasi atau tracking pihak ketiga

3. Tujuan Pengumpulan Data

Data dikumpulkan dan diproses semata-mata untuk:

  1. Mengelola operasional administrasi pondok pesantren
  2. Mencatat prestasi akademik dan absensi santri
  3. Mengirim notifikasi kegiatan, pengumuman, dan tagihan ke wali santri / pengguna
  4. Mengelola transaksi keuangan internal pondok
  5. Memungkinkan ekspor laporan PDF / Excel untuk arsip pondok
  6. Memenuhi kewajiban audit internal pesantren

Data TIDAK dijual, disewakan, atau dibagikan kepada pihak ketiga untuk tujuan komersial.

4. Penyimpanan & Keamanan

Semua data disimpan di layanan Firebase Cloud milik Google LLC dengan lokasi server di Asia (Asia-Southeast2 — Jakarta atau region terdekat). Layanan Firebase yang kami gunakan:

  • Cloud Firestore — basis data utama (santri, guru, prestasi, keuangan). Komunikasi via HTTPS terenkripsi.
  • Firebase Authentication — pengelolaan login dengan email/password atau Google Sign-In. Password di-hash di server Google, bukan disimpan dalam bentuk plain text.
  • Firebase Cloud Storage — penyimpanan logo, foto profil, gambar lampiran, tanda tangan digital. Akses dibatasi via aturan keamanan (Storage Rules).
  • Firebase Cloud Messaging (FCM) — pengiriman notifikasi push ke perangkat terdaftar.
  • Firebase Hosting — penyimpanan dan distribusi aplikasi web (kode HTML/CSS/JS).

Akses data dibatasi berdasarkan peran (role-based access control) melalui firestore.rules:

Peran Hak Akses
Santri Hanya bisa melihat data dirinya sendiri
Guru Akses data santri di lembaga yang diampu
Admin Pengurus Akses penuh data akademik & pengurus
Admin Keuangan Akses tambahan ke modul keuangan
Super Admin Akses penuh + manajemen pengguna

Komunikasi antara Aplikasi dan server menggunakan koneksi terenkripsi (TLS 1.2+). Password pengguna tidak pernah disimpan dalam bentuk plain text — kami menggunakan mekanisme hashing bawaan Firebase Authentication.

5. Pembagian dengan Pihak Ketiga

Kami HANYA membagikan data dengan penyedia layanan berikut, terbatas pada kebutuhan teknis:

Layanan Tujuan Kebijakan Privasi
Google LLC (Firebase) Hosting database, autentikasi, storage, notifikasi firebase.google.com/support/privacy
Google LLC (Play Store) Distribusi aplikasi Android policies.google.com/privacy

Kami TIDAK menggunakan platform iklan, analytics pihak ketiga (Google Analytics, Mixpanel, Facebook Pixel, dll.), atau layanan tracking lainnya.

6. Pengguna di Bawah Umur (Santri)

Sebagian besar santri pondok pesantren berusia di bawah 18 tahun. Sesuai dengan ketentuan Children's Online Privacy Protection Act (COPPA) dan UU Perlindungan Data Pribadi Indonesia (UU PDP No. 27/2022):

  • Akun santri di bawah umur dibuat oleh pengurus / wali santri, bukan oleh santri sendiri tanpa pengawasan.
  • Wali santri dianggap memberikan persetujuan pengelolaan data anak melalui pendaftaran santri ke pondok pesantren.
  • Wali santri berhak meminta akses, koreksi, atau penghapusan data anak dengan menghubungi Pengelola di kontak yang tertera.

7. Hak Pengguna

Sesuai UU PDP Indonesia, setiap pengguna berhak untuk:

  • Akses — melihat data pribadi yang kami simpan
  • Koreksi — meminta perbaikan data yang tidak akurat
  • Penghapusan — meminta penghapusan akun dan data terkait
  • Pencabutan persetujuan — menarik kembali persetujuan kapan saja
  • Portabilitas data — meminta ekspor data dalam format yang dapat dibaca
  • Pengaduan — mengajukan keluhan ke otoritas perlindungan data

Permintaan dapat diajukan melalui email lexanoisgroup3@gmail.com dengan subjek "Permintaan Data Pribadi — Ammu Online". Kami akan merespons dalam waktu maksimal 14 hari kerja.

8. Cookies & Penyimpanan Lokal

Aplikasi menggunakan:

  • LocalStorage — menyimpan preferensi tema (dark/light mode), bahasa, dan token sesi login agar pengguna tidak perlu login berulang.
  • IndexedDB / Cache — menyimpan data sementara untuk mode offline (PWA Service Worker).

Data ini disimpan hanya di perangkat pengguna, tidak dikirim ke server kami, dan dapat dihapus kapan saja melalui pengaturan browser atau menghapus data aplikasi.
